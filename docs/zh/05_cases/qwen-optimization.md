# Qwen 系列模型训推与算子优化

这是 hello-cann 第一阶段主线案例。

## 目标

- 在昇腾 NPU 上完成 Qwen 小模型到中等规模模型的推理部署。
- 基于同一模型完成单卡 LoRA 微调和推理验证。
- 使用 profiling 找到推理链路中的热点或可融合位置。
- 选择一个适合教学的算子点，完成 Ascend C 自定义算子开发。
- 将自定义算子接入 PyTorch / `torch_npu` 推理流程，比较优化前后的性能和正确性。

## 章节串法

1. 环境章节：检查 NPU、CANN、`torch_npu`，记录版本。
2. 推理章节：跑通 Qwen 小模型，再按机器条件记录更大模型结果。
3. 微调章节：完成单卡 LoRA 微调，验证模型输出变化。
4. profiling 章节：定位推理热点，解释相关组件在问题链路中的位置。
5. 算子章节：从简化版算子开始，再接入模型链路。
6. 总结章节：整理性能记录、代码结构、问题清单和复现实验表。
